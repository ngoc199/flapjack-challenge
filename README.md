# Flapjack Challenge

This project is built for the challenge of [FlapJack](https://docs.google.com/document/d/1AXSY1uI32keb0Pg13_oxU4eC83N67ierDasIpMssGPk).

## Available Commands

### Run the development server

```bash
yarn dev
```

### Build the project

```bash
yarn build
```
